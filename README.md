# Real-Time E-Commerce Analytics & ML Pipeline

## 🚀 Project Overview
This project builds a production-level data pipeline using Apache Spark.

## 🧱 Tech Stack
- PySpark
- Spark SQL
- Parquet
- Structured Streaming (coming soon)

## 📊 Features
- Batch ETL pipeline
- Data cleaning and transformation
- Partitioned Parquet storage
- Multi-table joins
- Feature engineering (upcoming)
- Machine learning pipeline (upcoming)
